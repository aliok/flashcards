Options

You can run this application offline. If you access this application from your browser, just bookmark the page and you can open it when you're offline. If you access this application within the Android app, Chrome web store app or IOS web app, the application can run offline already.

This application runs perfect with Android 2.2+ devices and IOS devices. For more information, please visit here.

There is also an application available on Android market. If you're on Android, I recommend you to install it instead of using the application within the browser.

Furthermore, a Chrome Web Store application is available here. You can install it on your Google Chrome.



If you think you're done with current word set, you can always fetch the next set manually.

You don't need to fetch sets manually at all, because the next set is automatically fetched when you went through all the words in current set.

A word set contains 100 entries and they're kept in your device. Thus, you can use the application offline (through your browser or the Android app).
